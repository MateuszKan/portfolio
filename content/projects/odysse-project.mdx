## Description

Odysse is a dynamic website developed from scratch, utilising Bootstrap for the frontend design and incorporating jQuery and PHPMailer to enhance functionality and user interaction.

## Features

- **Bootstrap Design**: A responsive layout built with Bootstrap.
- **PHPMailer Integration**: Email functionality through PHPMailer.
- **jQuery Scripting**: Enhanced user experience with jQuery.
- **Responsive Design**: Optimized for different devices and screen sizes.

## Technologies Used

- **Bootstrap**: A frontend framework for creating responsive and modern web designs.
- **jQuery**: A JavaScript library that simplifies event handling and animations.
- **PHPMailer**: A tool for sending emails through PHP.
