## Description

OhMyFood is a restaurant website created using HTML, CSS, and SASS, featuring a modern and responsive design that enhances user experience across various devices.

## Technologies Used

- **HTML**: The standard markup language for creating web pages.
- **CSS**: Used for styling the website.
- **SASS**: A preprocessor scripting language that extends CSS with features like variables and nested rules.
