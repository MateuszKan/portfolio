## Description

My portfolio showcases my projects and skills, built using Next.js, MDX, JavaScript, Shadcn, and Tailwind CSS, featuring a modern and responsive design.

## Technologies Used

- **Next.js**: A React framework for server-side rendering and static site generation.
- **MDX**: A format that allows you to write JSX embedded in Markdown.
- **JavaScript**: The core programming language used for dynamic behavior.
- **Shadcn**: A utility-first design system for building modern UIs.
- **Tailwind CSS**: A utility-first CSS framework for creating custom designs.
