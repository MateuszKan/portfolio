## Description

Ravenous is a restaurant search application built with React.js, HTML, CSS, Tailwind CSS, and DaisyUI, featuring a modern and responsive design.

## Technologies Used

- **React.js**: A JavaScript library for building user interfaces.
- **HTML**: The standard markup language for creating web pages.
- **CSS**: Used for styling the application.
- **Tailwind CSS**: A utility-first CSS framework for creating custom designs.
- **DaisyUI**: A component library that enhances Tailwind CSS with pre-designed components.
